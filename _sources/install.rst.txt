.. _installation:

Installation
============

You can install the latest version of ``bkbit`` directly from PyPI using pip:

.. code-block:: bash

   $ pip install bkbit

Alternatively, you can install the latest version of ``bkbit`` from the source code on GitHub:

.. code-block:: bash

   $ git clone https://github.com/brain-bican/bkbit.git
   $ cd bkbit
   $ pip install .
